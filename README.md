\# Akestreamlit



Akestreamlit is a Streamlit-based data science and AI analysis project focused on data cleaning, visualization, model training, and post-analysis using real-world datasets.



\## Features

\- Data Cleaning

\- Data Visualization

\- Model Training

\- Post-Review \& Analysis

\- Streamlit Interface



\## Tech Stack

\- Python

\- Streamlit

\- Pandas

\- Matplotlib / Seaborn

\- Scikit-learn



\## How to Run



1\. Activate your virtual environment

2\. Install requirements

&nbsp;  pip install -r requirements.txt

3\. Run app

&nbsp;  streamlit run app.py
